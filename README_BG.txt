Качване в GitHub Pages:
1. Разархивирай ZIP файла.
2. Качи съдържанието в root на repository-то: index.html, app.js, manifest.webmanifest, service-worker.js, README_BG.txt и папка icons/.
3. В Settings -> Pages избери Deploy from a branch, Branch: main, Folder: /root.
4. След успешен deployment отвори HTTPS адреса на телефона през Chrome.
5. Избери Install app или Add to Home screen.

Важно:
Данните се съхраняват локално на устройството. След терен винаги прави експорт в CSV и/или JSON. JSON файлът е резервно копие, което може да се импортира обратно в приложението.
